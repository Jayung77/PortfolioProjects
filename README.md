# COVID-19 Tableau 대시보드

## 프로젝트 소개

SQL을 활용하여 코로나19 데이터를 조회·가공한 후 Tableau를 이용해
국가별 감염 현황과 사망률을 시각화한 프로젝트입니다.

## 사용 기술

- SQL Server
- Tableau Public
- Excel

## 주요 분석 내용

- 전 세계 확진자 및 사망자 집계
- 국가별 감염률 분석
- 대륙별 사망자 수 비교
- 월별 감염 추이 시각화
- Tableau 대시보드 제작
